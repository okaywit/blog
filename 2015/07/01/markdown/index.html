标题

一级标题 #内容

二级标题 ##内容

三级标题 ###内容

四级标题 ####内容

五级标题 #####内容
六级标题 ######内容

链接

eg: 首页

code: [首页](http://www.okaywit.com/ "okaywit")

eg: http://www.okaywit.com/

code: <http://www.okaywit.com/>

强调

eg: 斜体

code: *内容* 或 _内容_

eg: 加粗

code: **内容** 或 __内容__

代码

eg: 代码

code: `代码`
