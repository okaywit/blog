linux常用命令

日常使用的高频率linux命令汇总

rm

删除目录：rm -r 目录 (-r:递归删除该目录下文件，-f:强制删除)

tar

tar.gz文件：tar -xzf 文件名

find

查找目录： find /范围 -name '关键字' -type d
查找文件： find /范围 -name '关键字' -print
